Kiss-Ass: The Promotioning
=========
Controls:
Day 1:
  -Press appropriate arrow key when prompted
  
Night 1 and 2:
  -Space: jump

Day 2:
  -Space: jump
  -S: slide

Day 3:
  -When the boss is done speaking, repeat what he says by typing the first letter of each word in the given order ("Simon")
